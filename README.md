# Next.js Dashboard Learning Repo

This repository is my learning project for the official [Next.js Course](https://nextjs.org/learn).

It follows the dashboard app exercises from the course and is used to practice core Next.js concepts in a real project structure.

## What This Project Covers

- Next.js App Router
- Server and Client Components
- Data fetching patterns
- Routing and layouts
- Form handling and validation
- Loading states and skeleton UIs
- Authentication patterns

## Project Structure

The project is organized around the App Router layout:

- `app/` for routes, pages, and server logic
- `ui/` for reusable UI components
- `app/lib/` for utilities, data access, and shared types
- `public/` for static assets
